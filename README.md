# WPDP Coaching Video Pathway

A web application for Tennis Canada's tennis coaches to provide structured videos and checklists for player development.

## Features

- Landing dashboard with two main pathways:
  - **Choose a stage**: Direct access to coaching stages
  - **Help me place player**: Guided player placement wizard

## Getting Started

Simply open `index.html` in a web browser to view the landing dashboard.

## Design

The design is inspired by Tennis Canada's official website, featuring:
- Dark red color scheme (#8B0000, #6B0000)
- Clean, modern layout
- Professional typography
- Responsive design for all devices
